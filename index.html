Reglas para que una pagina sea responsiva o adaptativa


¿Qué es el diseño responsivo?

Es un enfoque en el diseño web para el usuario

¿Qué es el diseño adaptativo?

Forma para maquetas las páginas web, para que sean capaz de adaptarse a la pantalla de cualquier dispositivo.

Caracteristicas

-Conexión de red
-Tamaño de la pantalla
-Tipos de interacción
-Resolución gráfica

Reglas

1.- Diseñar con contenido real.
2.- Diseño "in-browser".
3.- Desarrollar una biblioteca de patrones.
4.- Hacerlo Universal.
5.- Optimizar el sitio.
